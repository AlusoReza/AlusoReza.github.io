## Sesión 79 — Fix: dynamic name layout breaks animations

### Prompt
El cambio anterior (nombre dinámico) rompió las animaciones de resize. El nombre se teletransporta arriba a la derecha del contenedor y empieza a temblar en vez de quedarse en su sitio.

### Root cause
Dos FLIP independientes peleaban por el mismo elemento (`name`): el name FLIP (layout inline) y el row FLIP (layout horizontal) ambos seteaban `style.transform` y `style.transition` en el nombre, causando conflictos. El `transition: none` del row FLIP mataba la transición CSS de `flex-direction`.

1. Eliminar el name FLIP separado — la transición CSS en `flex-direction` maneja la animación del layout del nombre
2. Usar `.mobile-profile-text` (contenedor) en vez de `.mobile-profile-name` (hijo) para el row FLIP — el contenedor se anima por FLIP, el nombre se anima por CSS transition
3. El nombre ya no está en el array `elements` del row FLIP, así que nunca recibe `style.transform` ni `style.transition`

- `src/scripts/client.js` — Añadir `const text = mobileProfileInner.querySelector('.mobile-profile-text')`. Reemplazar `name` por `text` en el array `elements` del FLIP (líneas 602-615). Eliminar bloque name FLIP completo (líneas 586-603 originales).
